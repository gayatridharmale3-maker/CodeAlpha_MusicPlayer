# 🎵 Music Player

A modern and responsive **Music Player** built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

## 📌 Features

* 🎵 Play and Pause Music
* ⏮️ Previous and Next Song Controls
* 🖼️ Dynamic Album Cover Display
* 🎼 Song Title and Artist Display
* 📊 Interactive Progress Bar
* ⏱️ Current Time & Total Duration
* 🔊 Volume Control
* 📃 Playlist with Active Song Highlight
* ▶️ Auto Play Next Song
* ⌨️ Keyboard Shortcuts
* 📱 Fully Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Add your own MP3 audio files and update the paths in `script.js` if needed.
4. Open `index.html` in your browser.
5. Enjoy the Music Player.

## 📌 Note

This repository does **not** include MP3 files due to copyright restrictions. You can use your own audio files or royalty-free music and update the file paths in `script.js`.
